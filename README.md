# Play-to-Learn-AI

A collection of beginner-friendly reinforcement learning projects built around classic board games.

The goal of this project is to make reinforcement learning approachable through visual, interactive, and understandable environments rather than abstract mathematical examples.

## Repositories

### learn2ludo
Train AI agents to play Ludo using reinforcement learning, self-play, and simple decision strategies.

Focus areas:
- Dice-based stochastic environments
- Risk vs reward decisions
- Capturing and blocking strategies
- Visual learning progression
- Dice luck and probability statistics

### learn2mancala
Train AI agents to play Mancala through self-play and strategic resource optimization.

Focus areas:
- Planning ahead
- Resource distribution
- Tactical captures
- Long-term reward optimization
- Emergent strategy

---

## Project Goals

- Make reinforcement learning beginner-friendly
- Visualize learning and agent behavior
- Compare strategies across games
- Build educational AI environments
- Explore self-play and emergent behavior

---

## Planned Features

- Playable board game interfaces
- Human vs AI modes
- AI vs AI simulations
- Reinforcement learning agents
- Training visualizations
- Dice and probability statistics
- Game replay system
- Tournament simulations

---

## Technologies

- Python
- Pygame
- NumPy
- Matplotlib
- Gymnasium-compatible environments (planned)

---

## Philosophy

This project is intentionally designed to prioritize:
- clarity over optimization
- visibility over abstraction
- learning over performance

The environments are meant to help people understand how reinforcement learning agents make decisions and improve through experience.
